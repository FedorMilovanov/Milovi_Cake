# Milovi Cake — сайт кондитера

Премиальный статический сайт авторских тортов и десертов ручной работы — [milovicake.ru](https://milovicake.ru).

Без сборщика, без npm-зависимостей в runtime: чистый HTML / CSS / JS + один Python-скрипт для генерации страниц пригородов. Хостинг — GitHub Pages.

## Структура проекта


## Темы и визуальная система

- **Светлая тема:** кремовые поверхности (`#f5f0e8` / `#ede5d5`), глубокое золото (`#c9934a`) для интерактива, высокий контраст текста.
- **Тёмная тема:** off-black поверхности (`#1a1108` / `#1e1308`), слоистые карточки, мягкие золотые акценты (`#d4a76a`), без случайных светлых рамок и градиентов.
- **Иконки UI:** inline SVG; emoji допустимы только в пользовательских текстах (отзывы, сообщения мессенджеров).
- **Анимации:** только функциональные, с поддержкой `prefers-reduced-motion`.
- **Touch target:** не ниже 44 px на мобильных.
- **Защита от FOUC:** тема применяется inline-скриптом до рендера `<head>`.

## Пригороды — генератор страниц

Все 14 страниц пригородов собираются из одного шаблона. Редактировать сгенерированные `prigorody/<city>/index.html` руками **не нужно** — изменения будут перезаписаны при следующем `build.py`.

> ⚠️ Исключение: если в конкретной странице есть кастомные правки (например, иной inline-скрипт GA), сначала перенесите их в `_template.html`, а потом запускайте генератор.

**Изменить общий блок на всех страницах:**
1. Отредактируйте `prigorody/_template.html`.
2. Запустите генератор: `python3 prigorody/build.py`.

**Изменить данные конкретного города:**
1. Отредактируйте `prigorody/_cities.csv`.
2. Запустите `python3 prigorody/build.py`.
3. Если добавляли новый город — обновите `sitemap.xml`.

## Локальный запуск

```bash
python3 -m http.server 8080
# затем открыть http://localhost:8080
```

Альтернатива:

```bash
npx serve .
```

## Проверки перед деплоем

```bash
# Синтаксис JS
node --check js/main.js
node --check js/nav.js
node --check js/mc-2026.js
node --check js/gallery/main.js
node --check js/gallery/shuffle.js
node --check sw.js

# Генератор пригородов
python3 -m py_compile prigorody/build.py
```

Дополнительно вручную в браузере:

- главная: hero, каталог, калькулятор, отзывы, контакты;
- корзина, отправка в WhatsApp / Telegram / MAX;
- переключатель темы (мгновенный, без перезагрузки);
- мобильное меню, bottom nav, drawer корзины, lightbox;
- страницы пригородов;
- меренговый рулет, сертификаты, галерея;
- тёмная тема: проверить отсутствие светлых утечек в карточках и формах.

## Деплой

GitHub Pages: push в `main` запускает публикацию. Сборка не требуется. При обновлении CSS / JS не забудьте поднять номер `?v=...` в `index.html` и в массиве `PRECACHE` внутри `sw.js`.

## Изображения

- Основные форматы: WebP / AVIF.
- PNG используется только для иконок PWA (`icon-*.png`).
- Для патч-архива без медиа можно исключать `img/`, видеофайлы и `.git`.
