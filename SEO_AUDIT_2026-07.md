## 2. Что исправлено в этом коммите

### 2.1 🔧 Дыра во внутренней перелинковке (средний приоритет)
**Проблема:** 4 из 6 лендингов (`bento-torty`, `svadebnye-torty`, `detskie-torty`, `tort-s-dostavkoy`) имели блок `.lp-related` — перекрёстные ссылки на смежные категории. А `zakazat-tort-spb` и `tort-na-den-rozhdeniya` его **не имели**. Это ослабляло переток внутреннего веса и тематическую кластеризацию именно на двух самых коммерческих страницах.

**Исправление:** добавил блок `.lp-related` (класс уже был в `css/style.css`, новых файлов не создавал) на обе страницы, по образцу остальных лендингов, с корректными ссылками на смежные разделы (без ссылки на саму себя). Проверено: HTTP 200, JSON-LD валиден, разметка сбалансирована, dark-тема поддержана.

### 2.2 🔧 Обновлён `lastmod` (низкий приоритет)
Для двух изменённых страниц `lastmod` в `sitemap.xml` обновлён на `2026-07-11` — правдивый сигнал свежести для переобхода.

## 3. Рекомендации по приоритету (что ещё улучшить на сайте)

### 🟡 Средний приоритет

**3.1 Свежесть sitemap.** Все 28 URL имеют `lastmod: 2026-06-05` (месяц назад), при `changefreq: weekly`. Расхождение «обещанной» частоты и реальных дат снижает доверие краулера к `changefreq`. → Автоматизировать генерацию `lastmod` из `git log` даты последнего изменения файла (можно встроить в `scripts/audit.py` или отдельный шаг перед IndexNow).

**3.2 Тонкий контент на 2 индексируемых страницах.**
- `/gallery/` — ~119 слов. Для визуальной галереи ок, но добавить 150–200 слов вводного текста (что за торты, стили, как заказать по фото) усилит релевантность по «портфолио торты спб».
- `/prigorody/` (хаб) — ~95 слов. Добавить абзац-описание + краткий список городов текстом (не только карточки) — усилит хаб как посадочную по «доставка тортов пригороды спб/ЛО».

**3.3 Активировать видео-sitemap.** В галерее есть **16 реальных self-hosted `.webm`**, и `VideoObject`-схема на категориях уже ссылается на них с `contentUrl`/`thumbnailUrl`/`uploadDate`. При этом `sitemap-videos.xml` отключён (был шаблоном под VK/YouTube). → Сгенерировать реальный видео-sitemap из существующих `.webm` и раскомментировать строку `Sitemap` в `robots.txt`. Даст шанс на видео-выдачу в Яндексе и Google (рецепты/процесс сборки — популярный интент).

### 🟢 Низкий приоритет / «шлифовка»

**3.4 Уникальные OG-картинки по категориям.** Сейчас почти везде общий `og-cover.jpg`. Уникальная OG-картинка на `svadebnye-torty`, `detskie-torty`, `bento-torty` и т.д. повысит CTR при репостах в мессенджерах/соцсетях (важно для VK/Telegram-трафика в РФ). *(На `meringue-roll` и `prigorody` уже уникальные — образец есть.)*

**3.5 `meringue-roll` — заголовок 68 симв.** Чуть длиннее оптимума. Хвост `| Доставка по СПб и ЛО` частично дублирует «в СПб» из начала. Можно ужать до ~60 без потери ключа.

**3.6 robots.txt — директива `Host`.** `Host: milovicake.ru` — Яндекс отказался от неё ещё в 2018 (canonical теперь определяется 301-редиректами и `rel=canonical`). Не вредит, но бесполезна — можно убрать для чистоты.

**3.7 Гео-мета-теги (`geo.region`, `geo.placename`, `ICBM`).** Отсутствуют. Легаси-сигнал, современные Яндекс/Google опираются на `LocalBusiness`-схему (она уже есть с координатами), поэтому эффект минимальный — добавлять по желанию.

## 4. Off-site: главный резерв выхода в ТОП (важнее техники)

Техника уже не «держит» сайт — держат внешние факторы. По убыванию влияния для ниши «торты на заказ СПб»:

1. **Яндекс.Бизнес (Яндекс.Карты) — критично для РФ.** Убедиться, что карточка заполнена на 100%, есть фото, актуальные часы, категории «Кондитерская/Торты на заказ», раздел «Услуги» с ценами. Геопозиция в топе Яндекса по локальным запросам идёт в первую очередь из Бизнеса, а не из сайта.
2. **Отзывы — сейчас только 6 (rating 4.8).** Это узкое место и для сниппета (AggregateRating), и для доверия. Системно просить каждого клиента оставить отзыв на Яндекс.Картах и 2ГИС. Цель — 30+. Это самый быстрый рычаг роста конверсии и ранжирования в геовыдаче.
3. **2ГИС** — отдельно от Яндекса, отдельная карточка, свой трафик в СПб.
4. **Ссылки/упоминания:** свадебные площадки, каталоги кондитеров, локальные блогеры, коллаборации (фотографы, декораторы, ивент-агентства). Даже безанкорные упоминания бренда «Milovi Cake» полезны.
5. **Соцсети → сайт:** VK и Telegram с ссылкой на сайт и на конкретные лендинги (не только на главную).
6. **Поведенческие:** Яндекс сильно смотрит на CTR в выдаче и отказы. Хорошие title/description (уже есть) + быстрый сайт (уже есть) работают на это.
