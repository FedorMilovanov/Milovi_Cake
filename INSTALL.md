## Установка

1. **Бэкап текущих файлов** (на всякий случай):
   ```
   cp js/main.js js/main.js.broken_backup
   cp sw.js sw.js.backup
   cp gallery/index.html gallery/index.html.backup
   ```

2. **Замени файлы** на содержимое этого пакета.

3. **Деплой**.

4. **Hard refresh** в браузере (Ctrl+Shift+R) — обязательно, потому что Service Worker мог закешировать поломанный `main.js`. После hard refresh новый SW (`v2026.05.17-r19`) подтянет свежие версии.

5. **Проверь в DevTools (F12) → Console**:
   - НЕ должно быть `ReferenceError: addSliderTouch is not defined`
   - НЕ должно быть других красных ошибок

6. **Функциональная проверка** (5 минут):
   - В карточках товаров — слайды листаются стрелками и точками, есть автопрокрутка, на мобильном — свайпы пальцем
   - Калькулятор — переключается тип торта (бисквит → бенто → 3D), работают +/–, выбираются начинки
   - Бенто-карточка — переключается Стандарт/Макси
   - Отзывы — стрелки листают, переключаются вкладки
   - Лайтбокс — открывается клик по фото в калькуляторе и в карточках
   - Корзина — добавляются товары, удаляются, изменяется бейдж
