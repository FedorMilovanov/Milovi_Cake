## Установка

1. **Распакуй ZIP в корень репо** — он перезапишет файлы по правильным путям.

2. **Hard refresh в браузере** (Ctrl+Shift+R) — критически важно из-за Service Worker'а.

3. **Проверь визуально:**
   - **Галерея** (`/gallery/`): сетка карточек, кликаемые миниатюры, лайтбокс открывается красиво, иконки категорий разные (свадьба = сердце, бенто = коробочка, и т.д.)
   - **Калькулятор → бисквит** + наведение на начинку → должна появляться подсказка с описанием состава (КАК БЫЛО)
   - **Калькулятор → бенто** + наведение на начинку → подсказка **НЕ появляется** (как ты просил)
   - **Калькулятор → макси бенто** → то же
   - **Калькулятор → 3D-торт** → то же
   - **Декор** (Базовый / Авторский) → подсказка появляется (не трогали)
