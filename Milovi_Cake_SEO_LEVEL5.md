## 🔴 1. ПРОБЛЕМА №1: ПЕСОЧНИЦА ЯНДЕКСА ДЛЯ НОВОГО САЙТА

### 1.1 Факты о песочнице Яндекса (подтверждённые 2026)

- Яндекс **официально не признаёт** песочницу, но использует **возраст домена** как фактор ранжирования
- В **утекшем коде Яндекса** (2023) подтверждены факторы, связанные с возрастом домена и датой первой индексации
- Новые сайты проходят **«адаптационный период»** 2-9 месяцев
- Быстрый рост ссылочной массы = триггер фильтра («ссылочный взрыв»)
- Сайты на **бесплатном хостинге** или доменах 3-го уровня = гарантированная песочница

### 1.2 Статус Milovi Cake

| Параметр | Значение | Риск песочницы |
|----------|---------|---------------|
| Возраст домена | ~1 месяц (май 2026) | 🔴 Высокий |
| Хостинг | GitHub Pages (премиум, не бесплатный) | 🟡 Средний |
| Домен | milovicake.ru (платный .ru, свой) | 🟢 Низкий |
| Ссылочная масса | Естественная (нет взрыва) | 🟢 Низкий |
| Контент | Уникальный, качественный | 🟢 Низкий |
| SSL | ✅ HTTPS | 🟢 |
| Индексация | Частичная | 🟡 |
| Позиции | Низкие (скриншот) | 🔴 |

### 1.3 КАК УСКОРИТЬ ВЫХОД ИЗ ПЕСОЧНИЦЫ — 7 ПРОВЕРЕННЫХ МЕТОДОВ

#### Метод 1: Регистрация в Яндекс.Вебмастер + регион СПб
- Немедленно зарегистрировать сайт
- Указать регион «Санкт-Петербург»
- Отправить sitemap.xml
- Включить «Обход по счетчикам» (после установки Метрики)

#### Метод 2: Яндекс.Метрика — ОБЯЗАТЕЛЬНО
- Без Метрики Яндекс не видит реальные поведенческие факторы
- Вебвизор покажет, что пользователи проводят время на сайте (>2 мин с калькулятором)
- Это главный сигнал для выхода из песочницы

#### Метод 3: Плавный рост ссылочной массы
- НЕ покупать ссылки пачками (триггер «ссылочного взрыва»)
- 1-2 качественные ссылки в неделю
- Начать с каталогов (Яндекс.Бизнес, 2GIS, Zoon, Yell)
- Затем тематические (Wedding.ru, Gorko.ru)

#### Метод 4: Социальные сигналы
- VK группа с активностью
- Telegram канал
- Посты → трафик → поведенческие → выход из песочницы

#### Метод 5: Прямые заходы (Type-in Traffic)
- Попросить клиентов сохранять сайт в закладки
- Разместить ссылку в Instagram/VK bio
- Визитки, упаковка тортов — QR-код на сайт

#### Метод 6: Ускорение индексации
- IndexNow через GitHub Action
- Ручной переобход страниц в Вебмастере
- Свежий контент каждую неделю (блог)
- Обновление дат на страницах (lastmod)

#### Метод 7: Уникальный интерактивный функционал
- Калькулятор — ✅ УЖЕ ЕСТЬ (мощный сигнал)
- Корзина с WhatsApp/Telegram — ✅
- Добавить: квиз «Какой торт вам подходит?»

## 🔵 2. ПОВЕДЕНЧЕСКИЕ ФАКТОРЫ — ГЛАВНЫЙ РЫЧАГ ДЛЯ ЯНДЕКСА

### 2.1 Вес ПФ в ранжировании Яндекса (подтверждено утечкой 2023)

| Фактор | Вес | Порог |
|--------|-----|-------|
| CTR в выдаче | 0.8 | Высокий CTR → рост позиций |
| Dwell time (время на сайте) | 0.75 | >90 сек = положительно; >160 сек = сильно положительно |
| Глубина просмотра | 0.65 | >2 страниц |
| Last Click (конечный сайт) | 0.7 | Пользователь завершил поиск на сайте |
| Pogo-sticking (возврат в поиск) | -0.6 | Отрицательный сигнал |
| Type-in (прямые заходы) | 0.65 | >40% прямого трафика = буст «brand authority» +0.3 |
| Возвраты в течение 24ч | -0.6 | Сильно отрицательный |
| Время до первого клика | 0.5 | Быстрый клик по сниппету = хорошо |

### 2.2 Milovi Cake: оценка текущих ПФ

| Метрика | Прогноз | Комментарий |
|---------|---------|------------|
| Время на сайте | 🟢 >2 мин | Калькулятор + галерея + отзывы удерживают |
| Глубина | 🟢 2-3 страницы | Каталог → калькулятор → корзина |
| Отказы | 🟢 Низкие (<40%) | Пользователи взаимодействуют |
| Last Click | 🟡 Средний | Часть уходит в WA/TG, но это целевое действие |
| CTR | 🔴 Низкий | Нет расширенных сниппетов = низкая кликабельность |
| Type-in | 🔴 ~0% | Новый сайт, нет постоянных клиентов |
| Pogo-sticking | 🟡 Неизвестно | Без Метрики не измерить |

### 2.3 План улучшения ПФ

| Действие | Эффект | Срок |
|----------|--------|------|
| JSON-LD Schema (все 7 типов) | +30-40% CTR | 2-4 нед |
| Улучшенный Title + Description | +10-15% CTR | 1-2 нед |
| Яндекс.Метрика | Данные для доказательства ПФ | 1 нед |
| Внутренняя перелинковка | +глубина просмотра | 2-4 нед |
| Квиз «Какой торт подходит?» | +60-90 сек dwell time | 4-8 нед |
| QR-коды на упаковке → сайт | +type-in трафик | Постоянно |
| VK/Telegram → ссылки на сайт | +прямые заходы | Постоянно |
