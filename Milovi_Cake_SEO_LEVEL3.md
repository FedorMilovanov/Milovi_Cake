## 📈 4. CRO — CONVERSION RATE OPTIMIZATION

### 4.1 Воронка

SERP → Лендинг → Калькулятор → Корзина → WhatsApp/Telegram → Заказ

### 4.2 Точки трения и решения

| Проблема | Решение |
|---------|--------|
| Нет rich snippets | JSON-LD Schema (+30-40% CTR) |
| Слабый CTA | Усилить «Выбрать торт» |
| Сложный калькулятор | Упростить интерфейс |
| Неясность в корзине | Микро-текст «Далее: связь» |
| Только WA/TG | Форма обратного звонка |

### 4.3 Trust Signals

Счётчик отзывов, динамический рейтинг, 3-4 кейса с фото

### 4.4 CRO-гипотезы

- Sticky-bottom CTA на мобильных
- Таймер доставки
- Телефон в sticky-header
- Видео-отзыв вместо скриншота
- Калькулятор с 3 шагами
